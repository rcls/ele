This is the design for a digital thermometer using an HC08 processor and a
Nat. Semi SPI temperature sensor.  It drives an LCD via a few 4094 shift
registers.

Designs were done using gEDA/gschem/pcb.

lcd.sch - gschem circuit diagram.
lcd.pcb - gEDA pcb design.
lcd-flash.mem - firmware.  Who needs an assembler?
